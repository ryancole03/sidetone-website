# Sidetone Website

Landing page for [Sidetone](https://github.com/ryancole03/sidetone) - Real-time microphone passthrough with noise suppression for Windows.

## Links

- **Live Site**: https://ryancole03.github.io/sidetone-website/
- **App Repository**: https://github.com/ryancole03/sidetone
- **Download**: https://github.com/ryancole03/sidetone/releases/latest

## Tech Stack

- Pure HTML/CSS (no build required)
- GitHub Pages for hosting
- GitHub Actions for CI/CD
